# Upload your images here

Place all website images in this folder:

- `hero` backgrounds
- property photos
- agent photos
- logos

## Example file names

- `hero-main.jpg`
- `property-1.jpg`
- `agent-emma.jpg`
- `logo-partner-1.png`

## How to use in code

Use the path from `/public`:

- `/images/hero-main.jpg`
- `/images/property-1.jpg`

In Next.js components, this works directly with `next/image` or CSS background URLs.
