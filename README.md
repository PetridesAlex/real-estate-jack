# Real Estate Jack

Production-ready responsive real-estate website built with Next.js, TypeScript, and Tailwind CSS.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Install

```bash
npm install
```

## Run (development)

```bash
npm run dev
```

Open: `http://localhost:3000`

## Routes

- `/` Home
- `/properties` Property list with filters
- `/properties/[id]` Property details
- `/services`
- `/about`
- `/contact`
